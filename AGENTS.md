# AGENTS.md

このファイルは、WARP (warp.dev) がこのリポジトリで作業する際のガイダンスを提供します。

## プロジェクト概要

作業書作成支援ツールのスタンドアロンHTML型Reactアプリケーション集です。各HTMLファイルはビルドシステム不要の完全なシングルページアプリケーションで、CDN経由のライブラリのみに依存しています。

## 技術スタック

- **React 18** (unpkg経由CDN)
- **Babel Standalone** (ブラウザ内JSXトランスパイル用)
- **Tailwind CSS** (CDN)
- **UI言語**: 日本語

## ファイル説明

- `manual.html` - フル機能版: 動画キャプチャ + 画像アップロード、注釈(矢印、丸、テキストラベル、番号スタンプ)、画像操作、JSON保存/読込、PDF印刷
- `imgmanual.html` - 画像専用版: 動画キャプチャを除く同様の機能
- `newmanual.html` - 代替/実験版

## 開発方法

**ビルド不要。** HTMLファイルをブラウザで直接開くだけで実行できます。

変更をテストするには:
```
start manual.html
```

## アーキテクチャ

- ReactコードはすべてHTMLファイル内の`<script type="text/babel">`タグにインラインで記述
- 状態管理はReact Hooks (`useState`, `useRef`, `useEffect`) を使用
- アイコンはカスタムSVGコンポーネント (外部ライブラリ不使用)
- 印刷スタイルはCSS `@media print` ルールで制御
- データ永続化はブラウザでのJSONファイルダウンロード/アップロード

## 編集時に維持すべき重要機能

- 注釈システムは相対位置(パーセンテージベース座標)を使用
- 画像操作(パン/ズーム)はtransformベースの位置指定を使用
- 印刷レイアウトは横並び(row)と縦並び(col)の両方のステップレイアウトに対応
- 未保存変更時のページ離脱警告機能
